# stop-tracking.ch

Digital zine on AI, state surveillance, and digital self-defense.

## Development

```bash
npm install
npm run dev
```

## Build and test

```bash
npm run build
npm test
npm run lint
```

## Main content files

- Zine content: `src/content/zinePages.ts`
- Toolkit and links: `src/data/toolkit.ts`
